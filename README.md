# build

1. build it
   
   ```bash
   # get source code
   git clone git@github.com:fiefdx/MessageEncrypt.git
   cd ./MessageEncrypt
   # build it
   tns add android
   ./build.sh
   # run it with emulator
   tns run android
   
   ```
